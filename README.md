Sucursales
Gestión multi-sede para talleres con más de una ubicación física. Permite operar de forma centralizada manteniendo la separación por sucursal.

- Alta y configuración de sucursales — Registro de cada sede con nombre, dirección, datos fiscales y configuración propia.
  Cada sucursal puede tener su propio punto de venta AFIP.
- Stock por sucursal — El inventario se gestiona de forma independiente por sede. Cada sucursal ve y consume su propio stock.
  
- OTs por sucursal — Las órdenes de trabajo se crean y gestionan dentro de la sucursal correspondiente. Los reportes pueden verse por sede o consolidados.
  
- Transferencia de repuestos entre sucursales — Movimiento de stock de una sede a otra con registro del origen, destino, cantidad y responsable.
  
- Reportes por sucursal — Facturación, cobros, OTs y productividad desglosados por sede. Permite comparar el desempeño entre sucursales.
  
- Caja por sucursal — Registro de movimientos de efectivo de cada sede por separado. Apertura, cierre y arqueo de caja diario.
